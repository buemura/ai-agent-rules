# AI Agent Development Rules

A comprehensive repository of guidelines and best practices for AI agents to follow during software development, focusing on code quality, performance, observability, and maintainability.

## 📋 Overview

This repository serves as a centralized source of truth for development standards that guide AI agents (such as Claude Code, Claude Haiku, or other AI development tools) when working on projects. These rules ensure consistent, high-quality code output that aligns with engineering best practices.

### Why This Matters

- **Consistency**: Maintains uniform code style and architecture across projects
- **Quality**: Enforces performance and security best practices
- **Observability**: Ensures code is monitorable and debuggable in production
- **Onboarding**: Helps new team members understand development standards
- **Automation**: Provides clear directives for AI-powered development workflows
